# Content Fragment Selector Properties

The following table lists the properties that you can use to customize and leverage the Content Fragment Selector and how it is being rendered.

| Property                 | Type                                                                    | Required | Default         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|--------------------------|-------------------------------------------------------------------------|----------|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `imsToken`               | string                                                                  | No      |                 | IMS token used for authentication                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `repoId`                 | string                                                                  | Yes     |                 | Repository ID used for authentication                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `orgId`                  | string                                                                  | Yes     |                 | Organization ID used for authentication                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `locale`                 | string                                                                  | No      |                 | Locale data                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `env`                    | [Environment][Environment]                                              | No      |                 | Content Fragment Selector deployment environment                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `filters`                | [FragmentFilter][FragmentFilterURL]                                     | No      |                 | Filters to be applied for the list of content fragments. By default, fragments under `/content/dam` will be displayed. Default value: `{ folder: "/content/dam" }` |
| `isOpen`                 | boolean                                                                 | Yes     | `false`         | Flag to trigger opening/closing the selector                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `onDismiss`              | () => void                                                              | Yes     |                 | Function to be called when the dismiss button is pressed                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `onSubmit`               | ({ contentFragments: `{id: string, path: string}[]`, domainNames: `string[]` }) => void | Function to be called when the select button is pressed upon selecting one or multiple content fragments. The function will receive the selected content fragments with `id` and `path` fields and the domain names related to the repo's program id and environment id and which have the status `ready` and the tier `publish`. If there are no domain names it will be used the publish instance as a fallback domain. |
| `theme`                  | "light" | "dark"                                                        | No      |                 | Theme of the Content Fragment Selector. Default theme is set to the UnifiedShell environment theme                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `selectionType`          | "single" | "multiple"                                                   | No      | `single`        | Selection type can be used to restrict selection for the FragmentSelector                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `dialogSize`             | "fullscreen" | "fullscreenTakeover"                                     | No       | `fullscreen`   | Optional prop to control the dialog size                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `waitForImsToken`        | boolean                                                                 | No       | `false`        | Indicates whether the Content Fragment Selector is rendered in the context of SUSI flow and needs to wait for imsToken to be ready                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `imsAuthInfo`            | [ImsAuthInfo][ImsAuthInfo]                                              | No       |                | Object containing logged in user's IMS authentication information                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `runningInUnifiedShell`  | boolean                                                                 | No       |                | Indicates whether the Content Fragment Selector is running under UnifiedShell or standalone                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `readonlyFilters`        | [ResourceReadonlyFiltersField]                                          | No       |                | Readonly filters that can be applied for the list of content and cannot be removed. |
